# ZOV-INATOR
Небольшой hook для windows, который заменяет нажатие букв на другие, а именно:

| КЛАВИША      | СКАН-КОД           |
|:------------:|:------------------:|
| "з", "Z"     | "80", "90"         |
| "о", "O"     | "74", "79"         |
| "в", "V"     | "68", "86"         |

> Да, это жёстко.
> Собираем командой ниже!

### Сборка
```bash
g++ -o main.exe Main.c
```
